## 🔄 자동화 가이드

### 'ㄱ' 루프 사용법

```bash
# 기본 사용
ㄱ          # 다음 작업 자동 진행

# 병렬 실행
ㄱㄱ        # 2개 작업 병렬
ㄱㄱㄱ      # 3개 작업 병렬

# 상태 확인
ㄱ?         # 현재 상태 미리보기

# 긴급 핫픽스
ㄱ!         # 우선순위 무시하고 즉시 수정
```

### Hooks 자동화

- **SessionStart**: 환경 자동 설정, 진행 상황 표시
- **PreToolUse**: 민감 파일 보호, 위험 명령어 차단
- **PostToolUse**: 자동 포맷팅, 법률 체크, 타입 체크
- **Stop**: 다음 작업 자동 제안
