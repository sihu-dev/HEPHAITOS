| Plan | 월 가격 | 연간 가격 | 주요 기능 | 타겟 |
|------|---------|----------|----------|------|
| **Free** | ₩0 | - | 기초 교육, 백테스트 10회/월 | 신규 유저 |
| **Starter** | ₩9,900 | ₩99,000 | 전체 교육, 백테스트 100회 | 학습 중심 |

**2️⃣ 마켓플레이스 (25% 예상)**
- 전략 판매: 크리에이터 70% / 플랫폼 30%

- 템플릿/지표: ₩5,000 ~ ₩50,000

**3️⃣ 거래 수수료 (15% 예상)**
- 카피 트레이딩: 거래액 0.1~0.3%
- 자동매매 실행: 거래액 0.05%

### 📊 단위 경제학 (Unit Economics)

```
CAC (고객획득비용): ₩15,000
  - 소셜 광고: ₩10,000
  - 콘텐츠 마케팅: ₩5,000

LTV (고객생애가치): ₩450,000
  - ARPPU: ₩50,000/월
  - 평균 구독 기간: 9개월
  - Churn Rate: <5%/월

LTV:CAC = 30:1 ✅ (목표 >3)
Payback Period = 1.5개월 ✅ (목표 <12개월)
Gross Margin = 85% ✅ (SaaS 목표 >70%)
```
